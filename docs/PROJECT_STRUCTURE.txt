⚙️ BACKEND (backend/)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Location:    backend/lawAgent/
Tech:        TypeScript + Prisma + Supabase
Database:    PostgreSQL

Commands:
  cd backend/lawAgent && npm install    # Install
  npm run dev                           # Development
  npm run migrate                       # Migrations
  npm run seed                          # Seed DB

From root:
  npm run backend                # Start backend
  npm run backend:migrate        # Migrate
  npm run backend:studio         # Prisma Studio

🚀 GETTING STARTED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install Dependencies
   cd frontend && npm install
   cd ../backend/lawAgent && npm install

2. Setup Environment
   copy .env.example .env
   # Edit .env with your values

3. Setup Database
   cd backend/lawAgent
   npm run migrate
   npm run seed

4. Start Development
   Terminal 1: npm run frontend
   Terminal 2: npm run backend
